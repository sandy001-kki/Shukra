# Contributing

1. Fork the repository at [github.com/sandy001-kki/Shukra](https://github.com/sandy001-kki/Shukra).
2. Create a feature branch for your change.
3. Run `make generate` and `make manifests` before opening a pull request.
4. Run the test suite with `make test`.
5. Expect review on API compatibility, tenancy safety, observability, and
   reconciliation idempotency.
