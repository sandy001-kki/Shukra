# Support

Use these paths depending on what you need:

## Questions and usage help

- read the README first
- read `docs/cli.md`, `docs/bring-your-own-cluster.md`, and `docs/troubleshooting.md`
- open a GitHub Discussion or issue if the docs do not answer the question

## Bug reports

Use the bug report issue template and include:

- Shukra version
- cluster type
- relevant `shukra doctor` output
- relevant `AppEnvironment` status
- operator logs if available

## Security issues

Do not open public issues for vulnerabilities.

Use the process in `SECURITY.md`.
